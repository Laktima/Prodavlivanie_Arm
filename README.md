# Prodavlivanie_Arm
Расчёт на продавливание по СП 63.13330.2018 и отчёту НИИЖБ (Договор №709). Логика соответствует типовому расчёту «41. Пилон в осях» (без привязки к Excel). Усилия, отверстия, расчётный контур, поперечная арматура, подробный отчёт с формулами, экспорт DXF для AutoCAD.

## Описание файлов в папке

| Файл | Назначение |
|------|------------|
| **index.html** | Одна страница с вкладками «Обучение», «Пример», «Калькулятор». В калькуляторе: форма ввода, эскиз (вид сверху — колонна, контур U, отверстия, арматура), результат расчёта, кнопки «Скачать отчёт (.txt)» и «Скачать эскиз (DXF)». |
| **calc-data.js** | Справочники Rbt_coef, Rsw_MPa, diam_Asw. Источник данных — лист «Всп» типового расчёта; файл Excel не требуется. |
| **arm-placement.js** | Расстановка поперечной арматуры: функции `poperechkaCalc` и `getArmPlacement`. Формулы по методике (зона 1,5·h0, отступы от граней колонны). Используется в расчёте Fsw и при отрисовке схемы. |
| **calc.js** | Полный расчёт на продавливание по СП и отчёту НИИЖБ: периметр с учётом отверстий и края плиты, Fb,ult, Fsw, моменты, проверка по нескольким контурам, формирование текста отчёта. Вызов `runCalc(input)` без привязки к Excel. |

## Структура (кратко)
- **index.html** — интерфейс: вкладки, форма, эскиз, отчёт, экспорт.
- **calc-data.js** — справочные таблицы (Rbt, Rsw, Asw).
- **arm-placement.js** — координаты узлов сетки поперечной арматуры от граней колонны.
- **calc.js** — расчёт и текст отчёта.

## Запуск
Открыть `index.html` в браузере. Расчёт не требует сервера и не обращается к файлам Excel.

Порядок подключения скриптов
calc-data.js → poperechka-from-excel.js → arm-placement.js → punching-ub-holes.js → calc.js → app.js
